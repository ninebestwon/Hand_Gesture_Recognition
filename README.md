# Hand Gesture Recognition

## Dataset : Nina-Pro
* Number of Channel : 12
* Number of Class : 17
* Sampling Frequency : 2,000 Hz

## Model Training
* Multi-Layer Perceptron (MLP)
* Activation Function : ELU
* Optimizer : NAdam
* Dropout : True

## Inference
* Train Accuracy : 0.9505
* Test Accuracy : 0.7665
